# Taskflow Project
Proyecto inicial

Aplicación web (HTML + CSS + JavaScript) para gestionar tareas con persistencia en LocalStorage.

## 🚀 Demo
(pon aquí tu URL de Vercel)

## ✨ Funcionalidades
- Añadir tareas (texto, categoría y prioridad)
- Eliminar tareas
- Persistencia: las tareas se guardan en LocalStorage
- Filtro de búsqueda por texto
- Diseño responsive y modo oscuro

## 🧰 Tecnologías
- HTML semántico
- CSS (variables `:root`, Flexbox, Media Queries, Transiciones)
- JavaScript (DOM, eventos, LocalStorage)

## 📝 Uso
1. Escribe una tarea y pulsa **Añadir** o **Enter**
2. Filtra usando el buscador
3. Elimina con el botón **Eliminar**
