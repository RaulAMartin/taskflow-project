Axios

Axios es una librería para hacer peticiones HTTP desde frontend o backend. Simplifica el envío de datos, manejo de errores e interceptores frente a fetch.

Postman

Postman es una herramienta para probar APIs. Permite enviar peticiones HTTP, guardar colecciones y verificar respuestas y errores.

Sentry

Sentry es una plataforma de monitorización de errores. Se usa para detectar excepciones en tiempo real tanto en frontend como en backend.

Swagger

Swagger se usa para documentar APIs REST. Permite describir endpoints, parámetros, respuestas y probar la API desde una interfaz visual.
